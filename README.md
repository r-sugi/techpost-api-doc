# Skeleton project for Swagger


# install
```
npm install -g swagger
```

# run server
```
swagger project start techpost-api-doc
```

# docs page
```
http://localhost:10010/docs/
```

# edit swagger.yaml on browser
```
swagger project edit techpost-api-doc
```
